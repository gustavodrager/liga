# 🔧 Backend

## 1. Configuração

Usar a seção já existente:

```json
"WhatsappConvitesCadastro": {
  "Enabled": true,
  "AccountSid": "",
  "AuthToken": "",
  "RemetenteWhatsapp": "whatsapp:+14155238886",
  "UrlApp": "http://localhost:5173"
}
```

* ler via `IOptions`
* não hardcodar valores
* suportar variáveis de ambiente

---

## 2. Criar classe de configuração

```csharp
public class WhatsappConvitesCadastroConfig
{
    public bool Enabled { get; set; }
    public string AccountSid { get; set; }
    public string AuthToken { get; set; }
    public string RemetenteWhatsapp { get; set; }
    public string UrlApp { get; set; }
}
```

Registrar no `Program.cs` conforme padrão existente.

---

## 3. Criar contrato na aplicação

Criar interface:

```csharp
public interface IEnvioConviteWhatsappService
{
    Task EnviarAsync(string telefone, string nome, string token);
}
```

---

## 4. Implementação na Infraestrutura (Twilio)

Criar classe:

```csharp
public class TwilioWhatsappService : IEnvioConviteWhatsappService
```

Responsabilidades:

* inicializar Twilio com SID e Token
* montar link do convite
* montar mensagem
* enviar mensagem

### Montagem do link

```csharp
var link = $"{config.UrlApp}/convite/{token}";
```

### Mensagem

```text
Olá, {nome}!

Você recebeu um convite para acessar a Plataforma QuebraNunca.

Clique no link para concluir seu cadastro:
{link}
```

### Envio

* usar SDK oficial Twilio
* `from`: config.RemetenteWhatsapp
* `to`: $"whatsapp:{telefone}"

---

## 5. Regras importantes

* se `Enabled = false` → não envia
* erro no envio NÃO pode quebrar criação do convite
* erro deve ser tratado e registrado
* envio NÃO altera status do convite
* token continua sendo única fonte de verdade

---

## 6. Integrar no fluxo de convite

Localizar caso de uso de criação de convite.

Após persistir convite:

```csharp
if (config.Enabled && telefoneInformado)
{
    await envioWhatsappService.EnviarAsync(telefone, nome, token);
}
```

IMPORTANTE:

* não bloquear fluxo se falhar
* usar try/catch
* registrar erro se necessário
